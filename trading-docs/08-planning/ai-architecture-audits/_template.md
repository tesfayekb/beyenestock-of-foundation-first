## Purpose

This template will define the per-spec audit format used during 
Phase 1 of the AI architecture documentation work. It will cover:

- Audit dimensions: file/line references, schema references, commit 
  hashes, existing-system overlap, decision conflicts
- Three-AI roles: primary auditor (Cursor), cross-check auditor 
  (Claude), architectural validator (GPT-5.5 Pro)
- Three correction classes: mechanical, content, architectural intent
- Output format: 1-page redline per spec
- Resolution authority hierarchy

The full template will be drafted by Claude in Phase 1 P1.3 and 
reviewed by GPT-5.5 Pro and Cursor before adoption.
